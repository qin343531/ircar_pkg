# ircar_pkg
巡检车ros2框架
# 文件说明
## ircar_pkg
可见光摄像头TCP无线图传+手柄控制
## ircar_pkg_hk
在`ircar_pkg`的基础上添加了海康威视红外相机的rtsp流无线图传
## ros2_lidar
从商家给的例程中转为了ros2工程
### CamseneCSTool_D3YD.rar
这是windows平台的上位机
### SDK_V3.0-master.zip
内含有商家给的例程,基于cmake框架
## ros2_ydlidar
EAI YDLIADR X2激光雷达,有ROS,SLAM使用,官方资料较多
## ros2_ydlidar_SDK
需要进入这个压缩包的build目录使用make进行安装，才可以使用ros2_ydlidar包
